# DeskTransfer 使用说明

## 概述

DeskTransfer 是一个简单的局域网图片传输软件，支持批量图片传输功能。它由发送端和接收端两个部分组成，可以在局域网内快速传输图片文件。

## 系统要求

- Python 3.6 或更高版本
- 操作系统：Windows、macOS 或 Linux
- 局域网环境

## 安装步骤

1. 确保已安装 Python 3.6 或更高版本
2. 下载 DeskTransfer 项目文件
3. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```

## 使用方法

### 接收端设置

1. 在接收端电脑上运行接收端程序：
   ```
   python receiver.py
   ```

2. 程序启动后，会显示本机IP地址和端口号（默认为12345）

3. 点击"启动服务器"按钮开始监听连接

4. 服务器启动后，会自动创建一个以当前时间戳命名的接收文件夹

5. 等待发送端连接

### 发送端设置

1. 在发送端电脑上运行发送端程序：
   ```
   python sender.py
   ```

2. 在"接收端IP地址"输入框中输入接收端的IP地址

3. 点击"添加文件"按钮选择要发送的图片文件（支持批量选择）

4. 点击"连接"按钮连接到接收端

5. 连接成功后，点击"发送文件"按钮开始传输

6. 传输完成后，可以在接收端的接收文件夹中找到传输的图片

## 功能特点

- **批量传输**：支持一次性选择和传输多个图片文件
- **进度显示**：实时显示传输进度和状态
- **错误处理**：完善的错误处理和提示机制
- **跨平台**：支持 Windows、macOS 和 Linux 系统
- **简单易用**：直观的图形界面，操作简单

## 常见问题

### Q: 无法连接到接收端怎么办？

A: 请检查以下几点：
1. 确保两台电脑在同一局域网内
2. 确保接收端程序已启动并正在监听
3. 检查IP地址是否正确
4. 检查防火墙设置，确保端口12345未被阻止

### Q: 传输速度很慢怎么办？

A: 传输速度可能受以下因素影响：
1. 网络带宽和稳定性
2. 图片文件大小
3. 网络延迟

### Q: 可以传输其他类型的文件吗？

A: 当前版本仅支持图片文件传输，后续版本可能会支持更多文件类型。

### Q: 传输中断怎么办？

A: 如果传输中断，请重新连接并重新传输文件。当前版本不支持断点续传功能。

## 技术细节

### 网络协议

DeskTransfer 使用 TCP/IP 协议进行通信，默认使用端口 12345。通信协议基于 JSON 格式的消息，包含以下消息类型：

- 握手消息 (handshake)
- 文件信息消息 (file_info)
- 文件数据消息 (file_data)
- 文件传输结束消息 (file_end)
- 批量传输结束消息 (batch_end)
- 错误消息 (error)

### 安全性

当前版本未实现加密传输，请在安全的局域网环境中使用。敏感文件传输请谨慎使用。
