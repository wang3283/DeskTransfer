===============================================================================
                    DeskTransfer - 局域网图片传输工具
                        给Windows用户的简明说明
===============================================================================

【重要提示】
打包后的程序可以在没有安装Python的Windows电脑上直接运行！
无需安装任何软件，解压即用！

===============================================================================
                              快速开始
===============================================================================

如果你是程序使用者（不需要打包）：
----------------------------------------
1. 获得程序包（DeskTransfer文件夹或ZIP包）
2. 解压到任意位置（如果是ZIP包）
3. 双击运行 "start.bat"
4. 选择启动接收端(1)或发送端(2)
5. 按照界面提示操作

如果你是开发者（需要打包程序）：
----------------------------------------
1. 确保已安装Python 3.6或更高版本
2. 双击运行 "一键打包.bat"
3. 等待3-5分钟完成打包
4. 在 dist\DeskTransfer 目录获得可执行文件

===============================================================================
                            使用步骤详解
===============================================================================

【接收端操作】（在接收文件的电脑上）
----------------------------------------
1. 运行接收端程序：
   - 双击 start.bat，选择 1
   - 或直接双击 DeskTransfer-Receiver.exe

2. 启动服务器：
   - 点击"启动服务器"按钮
   - 记住显示的IP地址（例如：192.168.1.100）
   - 记住端口号（默认：12345）

3. 等待发送端连接...

4. 接收完成后：
   - 文件保存在 data\received 目录
   - 可以点击"打开接收文件夹"按钮


【发送端操作】（在发送文件的电脑上）
----------------------------------------
1. 运行发送端程序：
   - 双击 start.bat，选择 2
   - 或直接双击 DeskTransfer-Sender.exe

2. 配置连接：
   - 在"接收端IP地址"中输入接收端的IP地址
   - 端口号保持默认（12345）

3. 选择文件：
   - 点击"选择文件"按钮
   - 选择要传输的图片（可多选）
   - 支持的格式：JPG、PNG、GIF、BMP、TIFF、WebP

4. 开始传输：
   - 点击"连接"按钮连接到接收端
   - 连接成功后，点击"发送文件"
   - 等待传输完成

===============================================================================
                            常见问题
===============================================================================

问：程序无法启动怎么办？
----------------------------------------
答：1. 确认Windows版本（需要Windows 7或更高版本）
    2. 右键选择"以管理员身份运行"
    3. 如果杀毒软件报警，添加到白名单
    4. 可能需要安装Visual C++ Redistributable
       下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe


问：无法连接到接收端？
----------------------------------------
答：1. 确保两台电脑在同一局域网（同一WiFi或路由器）
    2. 检查IP地址是否输入正确
    3. 检查接收端服务器是否已启动
    4. 检查防火墙设置（允许程序通信）
    5. 尝试使用ping命令测试网络连通性


问：Windows防火墙拦截怎么办？
----------------------------------------
答：1. 首次运行时会弹出防火墙提示，选择"允许访问"
    2. 手动设置：
       - 控制面板 → Windows防火墙
       - 允许应用通过防火墙
       - 找到并勾选 DeskTransfer-Sender.exe 和 DeskTransfer-Receiver.exe
       - 允许专用网络和公用网络


问：杀毒软件报警怎么办？
----------------------------------------
答：这是正常的误报（因为程序是新打包的，没有数字签名）
    解决方法：
    1. 将程序添加到杀毒软件白名单
    2. 或暂时关闭杀毒软件的实时保护（传输完成后再开启）


问：传输中断了怎么办？
----------------------------------------
答：1. 检查网络连接是否稳定
    2. 重新启动程序并重试
    3. 确保没有其他程序占用端口12345


问：如何更改端口号？
----------------------------------------
答：在发送端和接收端的界面上都可以修改端口号
    确保两端使用相同的端口号


问：支持哪些文件格式？
----------------------------------------
答：目前支持常见的图片格式：
    - JPG/JPEG
    - PNG
    - GIF
    - BMP
    - TIFF
    - WebP

===============================================================================
                            技术信息
===============================================================================

程序版本：v1.0.0
开发语言：Python
打包工具：PyInstaller
GUI框架：Tkinter
通信协议：TCP/IP
默认端口：12345

系统要求：
- 操作系统：Windows 7 或更高版本
- 网络：局域网连接
- 防火墙：允许TCP端口12345
- 磁盘空间：至少50MB可用空间

文件大小：
- 发送端：约20-30MB
- 接收端：约20-30MB
- 完整包：约50-60MB（包含数据目录）

===============================================================================
                            高级功能
===============================================================================

【批量传输】
可以一次选择多个图片文件进行批量传输

【进度显示】
实时显示传输进度和速度

【日志记录】
程序会记录传输日志，方便查看历史记录

【自动保存】
接收的文件自动保存到 data\received 目录
文件名不会重复（自动添加时间戳）
